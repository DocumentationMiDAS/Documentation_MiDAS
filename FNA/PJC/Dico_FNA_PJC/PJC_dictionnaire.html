KCFPP
PJCCFER
CODE FERMETURE PJC
Code fermeture de la PJC pour motif fin de prise en charge. En cas de simultanéité d’apparition, la plus petite valeur sera privilégiée (cas de la fin d’ASC). Ce code est renseigné pour les trois types de PJC (KCPJC = ‘1’, ‘2’ ou ‘3’). Toute modification de cette valeur induit une rupture sur les PJC.
10. Fin de droit réglementaire :

- Limite d’âge à 65 ans

- Durée maximale atteinte

- Nouveau droit sur même prise en charge (sauf ACA)

- Date limite d’indemnisation

12. Fin période AER
15. Exclusion définitive DDTE ou Assédic (122 jours)
16. Exclusion définitive DDTE (autre)
20. radiation (reprise d’emploi, décès,…)
30. Suspension Assédic (non réponse à enquête au 122e jour)
40. Mise en attente de renseignements (ex: ressources RSN, déclaration DSM sauf attente de justificatif)
99. Sans code fermeture (en cours d’indemnisation, exclusion temporaire,…)
XX. Valeur hors table

KCOPP
PJCCOUV
CODE OUVERTURE PJC
Motif d’ouverture de la PJC suite à une décision de liquidation. Ce code est renseigné pour les trois types de PJC (KCPJC = ‘1’, ‘2’ ou ‘3’). Toute modification de cette valeur induit une rupture sur les PJC.
01. Admission en indemnisation
02. Réadmission en indemnisation
03. Admission au titre d’une aide
10. Transfert de l’indemnisation
20. Reprise de l’indemnisation après interruption du droit (sauf sur stage AFR ou RSP ou suite à stage AFR ou RSP)
30. Point de départ dans l’application (migration)
40. Révision du SJR/SJC suite à perte d’emploi conservé
50. Ouverture poursuite indemnisation sur antécédent exécutable
99. Sans objet (y compris pour une entrée AFR, ERF et RSP, et sur la reprise RAC/RSN qui suit le stage AFR ou RSP)
XX. Valeur hors table

KCPJC
PJCTYPE
CODE TYPE DE PJC
Cette donnée permet de connaitre le type d’une Période à Justification Constante
1. Période payée
2. Période non payée consommée :
- exclusions temporaires DDTE
- PND périodes imputables
- dépassement ressources plafond
3. Période non payée non consommée :
- Carence
- Différé
- Décalage
- Maladie
- Absence
- Périodes saisonnières
- PND périodes non imputables
- Suspensions non imputables
4. PJC payées au titre des AIDES (associé aux périodes de versement de l’ADR06, ASCRE06 ou ARCE06)
X. VALEUR HORS TABLE

KCPJCC
PJCTCAR
CODE TYPE PJC CARENCE
Cette rubrique définit le type de report. Toute modification de cette valeur induit une rupture sur les PJC. Ce code est renseigné pour les PJC de type ‘3’ et les différés en cours de droit.
01. Carence Préavis Payé non effectué (non géré)
02. CARENCE CONGES PAYES
03. CARENCE indemnités supra légales de rupture (ISLR)
04. DIFFERE D’INDEMNISATION
10. CARENCE CONGES PAYES ANNEXE 4 (<01/09/1992)
20. CARENCE CONGES PAYES ANNEXE 8
30. FRANCHISE ANNEXE 10 (non géré par AO)
40. CARENCE DELIBERATION 15
90. AUTRE TYPE DE CARENCE
99. SANS OBJET
XX. VALEUR HORS TABLE

KDDPJ
PJCDTDEB
DATE DEBUT PJC
C’est la date du premier jour de la période à justification constante, pendant laquelle les éléments de justification du CALCUL restent inchangés.

KDFPJ
PJCDTFIN
DATE FIN PJC
C’est la date du dernier jour de la période à justification constante, pendant laquelle les éléments de justification du CALCUL restent inchangés.

KNOPRP
PJCRD
RANG DEGRESSIVITE PJC
Indique le rang de dégressivité en cours sur la PJC. Ce code est renseigné pour les PJC payées (KCPJC = ‘1’). Toute modification de cette valeur induit une rupture sur les PJC.
00. Pas de dégressivité
01. Taux de dégressivité 1
02. Taux de dégressivité 2
03. Taux de dégressivité 3
04. Taux de dégressivité 4
05. Taux de dégressivité 5
06. Taux de dégressivité 6
07. Taux de dégressivité 7
08. Taux de dégressivité 8
09. Taux de dégressivité 9
99. Non déterminé

KQBPJP
PJCTAUX
MONTANT JOURNALIER ALLOCATION
Montant brut journalier de l’allocation. Cette rubrique est renseignée pour les PJC payées (KCPJC = ‘1’). Toute modification de cette valeur induit une rupture sur les PJC.

KQJSTP
PJCMITH
MONTANT NET JOURNALIER VERSE ITH
(MONTANT JOUR. INDEM TRANS. HERB.)
Montant net journalier versé au titre des ITH (Indemnités de Transport ou d’Hébergement. Ce code est renseigné pour les PJC payées de type ‘1’. Toute modification de cette valeur induit rupture sur les PJC.

KQRSPP
PJCSJRV
MONTANT SALAIRE JOURNALIER REVALORISE PLAFONNE
Montant SJR revalorisé plafonné. Cette rubrique est renseignée pour les PJC consommables (KCPJC = ‘1’, ou ‘2’). Toute modification de cette valeur induit une rupture sur les PJC.

KROD3
ODDIDENT
IDENTIFIANT DE L’ OUVERTURE DE DROIT
Référence de l’OD portant le fait générateur.
Clef de jonction avec KROD (ODDIDENT) du segment DROIT ODD - segment LN1S

KTDGAP
PJCCOEF
COEFFICIENT DEGRESSIVITE PJC
Il s’agit du coefficient de dégressivité appliqué lors du calcul d’un montant journalier dans la Nième période d’une durée dégressive en AUD. Le coefficient appliqué est égal au coefficient de la première, multiplié par lui-même autant de fois qu’il y a de périodes. Cette rubrique est renseignée pour les PJC payées (KCPJC = ‘1’).
